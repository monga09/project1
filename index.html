Project Overview

This process is to be used generate service contract quotes for Service First Customers word-wide.

The tool enables Asset Managers, on behalf of customers, to handle license renewals, asset installations, service contracts, and other lifecycle activities, ensuring optimized asset performance and uninterrupted service continuity.

Goal

Merge two existing tools into a single, unified experience to streamline input data management and reporting.

  • Role: UX Designer (Handled research, prototyping, testing, etc.)
  • Timeline: 5 months
  • Team: Designers, Business Analyst, Product Owners, Developers, Team Lead
  • Tools: Figma, Adobe Suite, Miro, HTML, CSS, Javascript

Problem Statement

Context: AMRR and ACAT were standalone tools with overlapping functions. Users needed to manage large sets of input data and run multiple reports.

Challenge:

  • Redundant workflows across both tools.
  • Difficulty sharing big data outputs across teams.
  • Reports were inconsistent and hard to track.

Goal: Create a single, integrated system that simplifies workflows, reduces duplication, and handles large-scale data reporting seamlessly.





Research

Methods: Stakeholder interviews, user workflow mapping, competitor/internal tool analysis.

Findings:

  • Users spent time switching between AMRR & ACAT.
  • Data-sharing bottlenecks caused delays in decision-making.
  • Reports were inconsistent across the two systems.
  • Personas/Users: Enterprise analysts, managers, reporting teams.

Ideation & Process

  • Explored integration models (merge vs. API-driven sync vs. new unified tool).
  • Created user journey maps to identify pain points in cross-tool workflows.
  • Prototyped different dashboard structures for managing big data inputs + outputs.




Design

  • Built wireframes showing:
  • Unified input panel (instead of entering the same data twice).
  • Consolidated reporting hub with filters for both AMRR & ACAT datasets.
  • Big data sharing module (download/export + role-based access control).
  • Ensured consistent Cisco design system usage for enterprise UX.




Final Solution

A single, integrated tool that merges AMRR & ACAT.

    Features:
  • Centralized input data hub.
  • Unified reporting interface.
  • Better & cleaner User Interface
  • Visually more appealing to end-users
  • Smooth big data sharing (CSV/Excel export, dashboard sharing, role-based access).




Impact & Results

  • Reduced duplication of effort by ~80% (no double data entry).
  • Improved collaboration: teams could share reports seamlessly.
  • Faster reporting cycles: turnaround time cut by (65%).
  • Positive user feedback: easier navigation, fewer steps.
